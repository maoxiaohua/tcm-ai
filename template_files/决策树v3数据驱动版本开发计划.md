### 阶段2: 后端API开发（第2周）

#### 2.1 数据转换服务

**文件**: `/opt/tcm-ai/services/decision_tree_data_service.py`

**核心功能**:
```python
class DecisionTreeDataService:
    def text_to_structured(text) -> dict
        """文本 → JSON"""

    def structured_to_text(data) -> str
        """JSON → 文本"""

    def structured_to_tree(data) -> dict
        """JSON → 树形"""

    def tree_to_structured(tree) -> dict
        """树形 → JSON"""

    def save_and_sync(id, content, source)
        """保存并同步两侧"""
```

#### 2.2 API路由

**文件**: `/opt/tcm-ai/api/routes/decision_tree_data_routes.py`

**端点**:
```
POST   /api/decision-tree-data/generate     # AI智能生成
POST   /api/decision-tree-data/save         # 保存并同步
GET    /api/decision-tree-data/{id}         # 获取数据
POST   /api/decision-tree-data/{id}/validate # 验证一致性
GET    /api/decision-tree-data/list         # 列表查询
DELETE /api/decision-tree-data/{id}         # 删除（软删除）
```

**产出**:
- 数据转换服务类
- 6个API端点
- 单元测试

### 阶段3: 前端改造（第3周）

#### 3.1 左侧文本编辑器改造

**功能**:
- 监听文本变化（防抖500ms）
- 自动保存到数据库
- 同步刷新右侧画布

**核心代码**:
```javascript
// 监听左侧变化
textEditor.addEventListener('input', debounce(async () => {
    await saveAndSyncFromText(textEditor.value);
}, 500));
```

#### 3.2 右侧画布改造

**功能**:
- 监听节点变化
- 自动保存到数据库
- 同步刷新左侧文本

**核心代码**:
```javascript
// 监听节点变化
canvas.on('nodeChanged', async (node) => {
    await saveAndSyncFromCanvas(canvas.exportStructure());
});
```

#### 3.3 AI生成增强

**功能**:
- 自然语言输入
- AI解析并生成标准格式
- 同时生成左侧文本和右侧树形

**核心代码**:
```javascript
async function aiGenerate(naturalText) {
    const result = await fetch('/api/decision-tree-data/generate', {
        body: JSON.stringify({natural_language: naturalText})
    });
    // 同时更新左右两侧
}
```

**产出**:
- 改造后的前端代码
- 实时同步功能
- AI智能生成功能
