# 多格式导出功能演示

## 🎯 核心改进

### 之前（JSON格式）
```json
{
  "disease_name": "头痛",
  "thinking_process": "...",
  "tree_structure": {...},
  ...
}
```
❌ 不易阅读，需要专业工具解析

### 现在（3种专业格式）

#### 📄 TXT格式示例
```
════════════════════════════════════════════════════════════
            特色诊疗方案
════════════════════════════════════════════════════════════

【方案名称】头痛

【创建时间】2025/10/27 10:26:00

【导出时间】2025/10/27 13:33:00

════════════════════════════════════════════════════════════
            一、诊疗思路
════════════════════════════════════════════════════════════

患者主诉头痛3天，伴随恶心、畏光症状。
需要详细询问头痛部位、性质、持续时间...

════════════════════════════════════════════════════════════
            二、决策树结构
════════════════════════════════════════════════════════════

节点总数：12
连接总数：11

────────────────────────────────────────────────────────────
【节点详情】
────────────────────────────────────────────────────────────

1. 初步问诊
   位置：(100, 50)
   类型：起始节点

2. 症状分析
   位置：(250, 50)
   类型：决策节点

3. 辨证论治
   位置：(400, 50)
   类型：治疗节点

────────────────────────────────────────────────────────────
【节点连接关系】
────────────────────────────────────────────────────────────

1. 初步问诊 → 症状分析
   连接标签：收集信息

2. 症状分析 → 辨证论治
   连接标签：确定证型
```

✅ 清晰的层次结构
✅ 易于打印和归档
✅ 任何文本编辑器都能打开

#### 📘 Word格式特点

```html
<h1>📋 特色诊疗方案：头痛</h1>

<table>
<tr><th>创建时间</th><td>2025/10/27 10:26:00</td></tr>
<tr><th>导出时间</th><td>2025/10/27 13:33:00</td></tr>
<tr><th>节点数量</th><td>12</td></tr>
</table>

<h2>一、诊疗思路</h2>
<div class="section">
患者主诉头痛3天，伴随恶心、畏光症状...<br>
</div>

<h2>二、决策树结构</h2>

<h3>2.1 节点信息</h3>
<table>
<tr><th>序号</th><th>节点内容</th><th>位置</th><th>类型</th></tr>
<tr><td>1</td><td>初步问诊</td><td>(100, 50)</td><td>起始节点</td></tr>
<tr><td>2</td><td>症状分析</td><td>(250, 50)</td><td>决策节点</td></tr>
</table>
```

✅ 专业的样式和颜色
✅ 表格清晰展示数据
✅ Word/WPS可直接打开编辑

```markdown
| 2 | 症状分析 | (250, 50) | 决策节点 |

### 2.2 连接关系

| 序号 | 起始节点 | 目标节点 | 连接标签 |
|------|----------|----------|----------|
| 1 | 初步问诊 | 症状分析 | 收集信息 |
| 2 | 症状分析 | 辨证论治 | 确定证型 |
```

✅ GitHub风格表格
✅ 支持版本控制
✅ 可转换为HTML/PDF

## 🔍 使用场景推荐

### 📋 医疗归档
推荐：**TXT格式**

- 快速查阅
- 最小存储空间

## 🎯 技术亮点

### 1. 结构化内容生成
```javascript
function generateTxtContent(pattern) {
    const divider = '═'.repeat(60);
    const subDivider = '─'.repeat(60);
    
    let content = `
${divider}
            特色诊疗方案
${divider}
...
    `;
    return content;
}
```

### 2. HTML格式Word文档
```javascript
function generateWordContent(pattern) {
    return `
<html xmlns:o='urn:schemas-microsoft-com:office:office'>
<head>
<style>
body { font-family: "Microsoft YaHei"; }
h1 { color: #1e40af; }
table { border-collapse: collapse; }
</style>
</head>
<body>
...
</body>
</html>
    `;
}
```

### 3. Markdown表格生成
```javascript
function generateMarkdownContent(pattern) {
    let content = `
| 序号 | 节点内容 | 位置 | 类型 |
|------|----------|------|------|
`;
    pattern.tree_structure.nodes.forEach((node, i) => {
        content += `| ${i+1} | ${node.text} | (${node.x}, ${node.y}) | ${node.nodeType} |\n`;
    });
    return content;
}
```
