### 2. 左侧文本编辑器自动保存 ✅

**功能流程**:
```
用户输入文本
    ↓
监听input事件
    ↓
500ms防抖延迟
    ↓
调用saveFromText()
    ↓
POST /api/decision-tree-data/save-from-text
    ↓
返回tree_structure
    ↓
调用updateCanvas()刷新右侧画布
    ↓
显示"✅ 已保存"状态
```

**技术细节**:
- ✅ 500ms防抖，避免频繁保存
- ✅ 支持input和paste事件
- ✅ 同步标志防止并发冲突
- ✅ 自动更新currentDecisionId
- ✅ 实时状态反馈

**代码示例**:
```javascript
onTextChanged() {
    this.updateSaveStatus('editing');
    clearTimeout(this.textSaveTimeout);

    this.textSaveTimeout = setTimeout(() => {
        this.saveFromText();
    }, 500);
}
```

```

    ↓
刷新右侧画布 (tree_structure)

**右→左同步**:
```

## 📋 使用指南

### 基本操作

1. **访问页面**:
   ```
   https://mxh0510.cn/static/decision_tree_v3_data_driven.html
   ```

2. **左侧文本编辑**:
   - 在"诊疗思路"文本框输入内容
   - 按照标准格式输入（【疾病】、【主症】等）
   - 停止输入500ms后自动保存
   - 右上角显示保存状态
   - 右侧画布自动更新

3. **右侧画布操作**:
   - 添加/移动/编辑节点
   - 停止操作500ms后自动保存
   - 右上角显示保存状态
   - （可选）左侧文本自动更新

4. **AI生成**:
   - 输入疾病名称
   - 点击"生成特色诊疗方案"
   - 等待AI生成（5-10秒）
   - 左右两侧同时更新
   - 自动保存到数据库

### 标准格式示例

```
【疾病】风寒感冒
【主症】恶寒发热（重）、头痛（中）、身痛（中）
【兼症】鼻塞、流清涕、咳嗽
【舌象】舌淡红，苔薄白
【脉象】脉浮紧
【证候】风寒表证
【处方】麻黄汤
【方剂】麻黄9克(君药) + 桂枝6克(臣药) + 杏仁6克(佐药) + 甘草3克(使药)
```

### 状态指示说明

- **✏️ 编辑中...**: 正在输入或修改
- **💾 保存中...**: 正在保存到数据库
- **✅ 已保存**: 保存成功（2秒后消失）
- **🤖 AI生成中...**: AI正在生成数据
- **❌ 保存失败**: 保存出错（3秒后消失）
