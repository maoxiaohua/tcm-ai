# 节点编辑自动覆盖诊疗思路问题修复

## 🐛 问题描述

### 用户反馈
用户在画布上编辑节点名称后（如将"处方1"改为"处方1+2"），发现：
1. ❌ 左侧诊疗思路被自动重新生成
2. ❌ 只显示一个分支的信息
3. ❌ 其他证候和处方的完整信息丢失
4. ❌ 出现错误识别（如"处方1+2"生成了"处方3"）

### 问题场景
```
步骤1: 用户手动编辑诊疗思路
------------------------------
主病：头痛
主证：

【证候1】脑袋瓜前面疼
病机：
症见：脑袋瓜前面疼
舌脉：
处方：处方1
加减：
  若因为喝酒过多：加 去痛片

【证候2】后脑疼
病机：
症见：后脑疼
舌脉：
处方：处方2

【证候3】侧面疼
病机：
症见：侧面疼
舌脉：
处方：处方3

步骤2: 用户在画布上编辑节点
------------------------------
将"处方1"节点名称改为"处方1+2"

步骤3: 保存后发现
------------------------------
❌ 诊疗思路被自动覆盖，只剩：

【证候1】脑袋瓜前面疼
病机：
症见：脑袋瓜前面疼
舌脉：
处方：处方1+2  ← 只保留了这个分支
加减：

❌ 处方2、处方3的内容全部丢失！
```

## 🔍 根本原因分析

### 代码流程
```javascript
// 用户双击节点编辑
function editNode(node) {
    showDialog();

    // 用户点击确认
    confirmBtn.onclick = function() {
        node.name = nameInput.value.trim();  // 更新节点名称
        updateNodeDisplay(node);

        // ❌ 问题根源：自动调用更新函数
        updateThinkingProcessFromNodes();  // 第4578行
    };
}
```

### 自动更新的问题
```javascript
function updateThinkingProcessFromNodes() {
    // 完全重新生成诊疗思路
    let thinkingText = '';

    // 遍历所有证候节点
    syndromes.forEach((syndrome, index) => {
        // 根据连接关系重新生成
        thinkingText += `【证候${index + 1}】${syndrome.name}\n`;
        // ...生成病机、症见、处方等
    });

    // ❌ 完全覆盖用户手动编辑的内容
    document.getElementById('doctorThought').value = thinkingText;
}
```

### 为什么会丢失内容？
1. **自动触发**：编辑任何节点名称都会自动调用更新函数
2. **完全重写**：函数会完全重新生成诊疗思路，不保留任何现有内容
3. **连接依赖**：只根据当前连接关系生成，如果连接不完整，就会丢失分支
4. **用户不知情**：用户以为只是改个节点名称，不知道会触发全局更新

## ✅ 解决方案

### 核心原则
**用户数据优先**：画布节点编辑不应该自动影响诊疗思路，用户完全控制何时更新

### 修改内容

#### 修改前（第4574-4579行）
```javascript
closeDialog();
showResult('成功', '✅ 节点信息已更新', 'success');

// ✨ 新增：自动更新诊疗思路
updateThinkingProcessFromNodes();  // ❌ 自动覆盖用户内容

// 移除ESC监听器
document.removeEventListener('keydown', escapeHandler);
```

#### 修改后（第4574-4582行）
```javascript
closeDialog();
showResult('成功', '✅ 节点信息已更新', 'success');

// ⚠️ 不自动更新诊疗思路，避免覆盖用户手动编辑的内容
// 用户可以通过点击"🔄 更新思路"按钮主动更新
console.log('💡 提示：节点已更新，如需同步诊疗思路，请点击"🔄 更新思路"按钮');

// 移除ESC监听器
document.removeEventListener('keydown', escapeHandler);
```

### 保留的功能
✅ **手动更新按钮**：用户可以主动点击"🔄 更新思路"按钮重新生成
✅ **确认对话框**：点击按钮前会弹出警告，防止误操作
✅ **专业格式**：手动更新时仍使用专业中医格式生成
