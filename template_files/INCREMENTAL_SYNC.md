# 节点名称增量同步功能

## 🎯 用户真实需求

### 核心诉求
> "我是想要我更新画布里面的节点的内容，同步得在左边的诊疗思路里面也能相应得同步更新。"

**期望行为**：
- ✅ 在画布上修改节点名称（如"处方1" → "处方1+2"）
- ✅ 左侧诊疗思路中对应的文字**自动同步更新**
- ✅ **只更新修改的部分**，其他内容完全保持不变
- ✅ 保留用户精心编辑的格式、段落、详细描述

## 🔧 技术实现

### 核心函数：syncNodeNameToThinking()

#### 位置
`/opt/tcm-ai/static/decision_tree_visual_builder.html` - 第2169-2205行

#### 完整代码
```javascript
function syncNodeNameToThinking(oldName, newName) {
    const thinkingTextarea = document.getElementById('doctorThought');
    if (!thinkingTextarea) {
        console.log('⚠️ 未找到诊疗思路输入框');
        return;
    }

    const currentText = thinkingTextarea.value;
    if (!currentText || currentText.trim() === '') {
        console.log('💡 诊疗思路为空，无需同步');
        return;
    }

    console.log('🔄 开始增量同步节点名称');
    console.log('  旧名称:', oldName);
    console.log('  新名称:', newName);

    // 使用全局替换，但保持文本结构不变
    // 这样可以替换所有出现的位置（可能在多个证候中引用同一个方剂）
    const updatedText = currentText.replace(
        new RegExp(escapeRegExp(oldName), 'g'),
        newName
    );

    if (updatedText !== currentText) {
        thinkingTextarea.value = updatedText;
        console.log('✅ 诊疗思路已同步更新');

        // 视觉反馈
        thinkingTextarea.style.borderColor = '#10b981';
        thinkingTextarea.style.boxShadow = '0 0 0 3px rgba(16, 185, 129, 0.1)';
        setTimeout(() => {
            thinkingTextarea.style.borderColor = '';
            thinkingTextarea.style.boxShadow = '';
        }, 1000);
    } else {
        console.log('💡 诊疗思路中未找到该节点名称，无需更新');
    }
}
```

### 辅助函数：escapeRegExp()

#### 位置
第2207-2210行

#### 代码
```javascript
function escapeRegExp(string) {
    return string.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
}
```

#### 作用
转义正则表达式特殊字符，确保精确匹配。

**示例**：
```javascript
escapeRegExp("处方1+2")  // 返回 "处方1\\+2"
escapeRegExp("黄芩(生)") // 返回 "黄芩\\(生\\)"
```

## 🐛 边界情况处理

### 1. 特殊字符处理
```javascript
// 示例：节点名称包含正则特殊字符
oldName = "处方1+2"
// 不使用 escapeRegExp 会出错
new RegExp("处方1+2")  // ❌ 错误：+ 是正则量词

// 使用 escapeRegExp 正确处理
new RegExp(escapeRegExp("处方1+2"))  // ✅ 正确：处方1\+2
```

### 2. 空值处理
```javascript
// 诊疗思路为空
if (!currentText || currentText.trim() === '') {
    return;  // 直接返回，不做任何操作
}
```

### 3. 未找到匹配
```javascript
// 替换后文本未变化
if (updatedText !== currentText) {
    // 只有真正替换了才更新
    thinkingTextarea.value = updatedText;
} else {
    // 未找到匹配，不做任何操作
    console.log('💡 诊疗思路中未找到该节点名称，无需更新');
}
```

### 4. 名称未改变
```javascript
// 编辑节点但名称没变
if (oldName !== newName) {
    syncNodeNameToThinking(oldName, newName);
}
// 如果相同，不调用同步函数
```
