# 特色诊疗方案管理功能测试指南

## 已完成的功能

### 1. ✅ 单个方案导出功能
**位置**: 每个诊疗方案右侧都有"导出"按钮

**功能**:
- 点击导出按钮可将该方案以JSON格式下载到本地
- 文件名格式: `特色诊疗方案_[疾病名称]_[时间戳].json`
- 包含完整的诊疗思路、决策树结构、临床模式等信息

**测试步骤**:
1. 打开 https://mxh0510.cn/static/decision_tree_visual_builder.html
2. 点击左侧"我的特色方案"按钮
3. 在弹出的历史列表中，每个方案右侧有蓝色"导出"按钮
4. 点击"导出"，自动下载JSON文件

### 2. ✅ 删除功能修复
**位置**: 每个诊疗方案右侧都有"删除"按钮

**功能**:
- 点击删除会弹出二次确认对话框
- 显示方案名称，提示"删除后将无法恢复"
- 删除成功后自动刷新列表

**测试步骤**:
1. 点击左侧"我的特色方案"按钮
2. 在历史列表中，每个方案右侧有红色"删除"按钮
3. 点击"删除"，确认对话框显示方案名称
4. 确认后，方案被删除，列表自动刷新

## API端点

### 删除方案
```
DELETE /api/delete_clinical_pattern/{pattern_id}
Headers: Authorization: Bearer {token}
```

**响应示例**:
```json
{
    "success": true,
    "message": "已成功删除诊疗方案：头痛",
    "data": {
        "pattern_id": "xxx-xxx-xxx",
        "disease_name": "头痛",
        "deleted_at": "2025-10-27T13:20:00"
    }
}
```

## 权限控制

- ✅ 只能删除自己创建的方案
- ✅ 管理员可以删除任何方案
- ✅ 删除前验证所有权
