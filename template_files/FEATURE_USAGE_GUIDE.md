# 特色诊疗方案管理功能使用指南

## 功能概览

本次更新为特色诊疗方案列表添加了两个重要功能：
- 📤 **导出功能**: 将单个诊疗方案导出为JSON文件
- 🗑️ **删除功能**: 安全删除不需要的诊疗方案

## 界面位置

访问 https://mxh0510.cn/static/decision_tree_visual_builder.html

点击左侧边栏的 **"我的特色方案"** 按钮，会弹出历史方案列表。

## 按钮说明

每个诊疗方案的右侧现在有两个按钮：

```
┌────────────────────────────────────────────────────┐
│ 头痛                                    [导出] [删除] │
│ 暂无诊疗思路描述                                    │
│ 创建时间：2025/10/27 10:26:00 | 节点数：12         │
└────────────────────────────────────────────────────┘
```

### 导出按钮 (蓝色)
- **颜色**: 蓝色背景 (#dbeafe)
- **功能**: 将该方案导出为JSON文件
- **文件名**: `特色诊疗方案_[疾病名称]_[时间戳].json`
- **内容包含**:
  - 疾病名称
  - 诊疗思路
  - 决策树结构
  - 临床模式
  - 医生专业知识
  - 导出时间

### 删除按钮 (红色)
- **颜色**: 红色背景 (#fee2e2)
- **功能**: 删除该诊疗方案
- **安全措施**:
  - 弹出确认对话框
  - 显示方案名称
  - 提示"删除后将无法恢复"
  - 只能删除自己创建的方案

## 使用流程

### 导出方案

1. 点击 "我的特色方案" 按钮
2. 在列表中找到要导出的方案
3. 点击蓝色 **"导出"** 按钮
4. 浏览器自动下载JSON文件

**提示**: 导出的JSON文件可以用于：
- 备份重要的诊疗方案
- 分享给其他医生参考
- 导入到其他系统

### 删除方案

1. 点击 "我的特色方案" 按钮
2. 在列表中找到要删除的方案
3. 点击红色 **"删除"** 按钮
4. 在确认对话框中点击 **"确定"**
5. 页面显示 "删除成功" 提示
6. 列表自动刷新，被删除的方案消失

**警告**: 删除操作不可恢复，请谨慎操作！

## 技术细节

### 权限控制
- ✅ 需要登录才能使用
- ✅ 只能删除自己创建的方案
- ✅ 管理员可以删除任何方案

### 导出格式 (JSON)
```json
{
  "disease_name": "头痛",
  "thinking_process": "患者主诉头痛...",
  "tree_structure": {
    "nodes": [...],
    "connections": [...]
  },
  "clinical_patterns": {...},
  "doctor_expertise": {...},
  "exported_at": "2025/10/27 13:20:00",
  "node_count": 12
}
```

### API端点
```
GET  /api/get_doctor_patterns/{doctor_id}  # 获取方案列表
DELETE /api/delete_clinical_pattern/{pattern_id}  # 删除方案
```

## 常见问题

### Q: 导出的JSON文件如何使用？
A: 可以用文本编辑器打开查看，或通过程序解析使用。

### Q: 删除后能恢复吗？
A: 不能。删除是永久性的，请确保不需要该方案后再删除。

### Q: 可以批量导出所有方案吗？
A: 当前版本仅支持单个导出。如需批量导出，请联系管理员。

### Q: 删除按钮是灰色的/点击无反应？
A: 请检查：
1. 是否已登录
2. 该方案是否是您创建的
3. 浏览器控制台是否有错误信息
