# 画布节点与诊疗思路自动同步功能说明

## ✨ 功能概述

当你在画布中编辑节点内容后，左侧的诊疗思路输入框会**自动同步更新**，确保内容始终保持一致。

## 🎯 解决的问题

### 之前的问题
```
1. 在画布中编辑节点 → 修改节点名称/描述
2. 点击"保存更改"
3. 左侧诊疗思路不变 ❌
4. 点击左侧"保存到思维库"
5. 保存的是旧内容 ❌
```

### 现在的流程
```
1. 在画布中编辑节点 → 修改节点名称/描述
2. 点击"保存更改"
3. 左侧诊疗思路自动更新 ✅
4. 点击左侧"保存到思维库"
5. 保存的是最新内容 ✅
```

```
分析：肝气郁结化火，肝阳上亢，风阳上扰清窍
```

## 🔧 技术实现

### 核心函数
```javascript
function updateThinkingProcessFromNodes() {
    // 1. 检查是否有节点
    if (nodes.length === 0) return;
    
    // 2. 按类型分组节点
    const nodesByType = {};
    nodes.forEach(node => {
        const type = node.type || 'other';
        if (!nodesByType[type]) {
            nodesByType[type] = [];
        }
        nodesByType[type].push(node);
    });
    
    // 3. 按顺序生成文本
    let thinkingText = '';
    // ... 处理各种节点类型
    
    // 4. 更新到输入框
    document.getElementById('doctorThought').value = thinkingText;
}
```

### 触发时机
```javascript
// 节点编辑保存时
document.getElementById('saveNodeEdit').onclick = function() {
    // 更新节点数据
    node.name = nameInput.value.trim();
    node.type = typeSelect.value;
    node.description = descTextarea.value.trim();
    
    // 更新显示
    updateNodeDisplay(node);
    
    // ✨ 自动同步诊疗思路
    updateThinkingProcessFromNodes();
};
```

## 💡 使用建议

### 场景1: 新建诊疗方案
```
1. 创建多个节点
2. 编辑第一个节点 → 保存 → 诊疗思路生成
3. 继续编辑其他节点 → 保存 → 诊疗思路累积更新
4. 所有节点编辑完成后，诊疗思路已完整
5. 直接点击"保存到思维库"
```

### 场景2: 修改已有方案
```
1. 打开历史方案
2. 修改某个节点内容
3. 保存节点 → 诊疗思路自动更新
4. 点击"保存到思维库" → 保存最新内容
```

### 场景3: 手动调整
```
1. 系统自动生成诊疗思路
2. 你可以在左侧输入框中手动调整
3. 添加个人见解和注释
4. 保存时保留你的修改
```

## ⚠️ 注意事项

### 1. 节点描述的重要性
- 节点名称会作为标题
- 节点描述会作为详细说明
- 建议填写详细的描述以获得更好的诊疗思路

### 2. 手动修改不会被覆盖（部分情况）
- 如果你手动修改了诊疗思路
- 再编辑节点并保存
- **诊疗思路会被重新生成**
- 建议：先完成节点编辑，再手动调整诊疗思路

### 3. 空节点不会显示
- 如果节点没有描述，只显示名称
- 空节点不影响整体结构

## 🚀 最佳实践

### 推荐工作流程

```
第三步：调整和完善

第四步：保存
- 点击"保存到思维库"

```
