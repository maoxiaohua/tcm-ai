# 自动化Git备份规则

## 自动备份文件类型
- *.html (前端文件)
- *.js (JavaScript文件)
- *.py (Python文件，除__pycache__)
- *.json (配置文件，除日志)
- *.md (文档文件)
- *.sql (数据库脚本)

## 排除文件类型
- __pycache__/*
- *.pyc
- *.log
- api/conversation_logs/*
- data/*.sqlite (运行时数据库)
- backup/*
- *cache*

## 工作流程
1. 修改前：git add <specific-file> && git commit -m "自动备份：修改前快照"
2. 修改后：git add <modified-file> && git commit -m "具体修复描述"
